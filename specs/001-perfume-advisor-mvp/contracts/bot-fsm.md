# Контракт FSM-потоков бота

**Дата**: 2026-04-27

Все диалоговые потоки ДОЛЖНЫ быть реализованы как `StatesGroup` в `perfume_bot/bot/states.py`.
Свободное хранение состояния только в Redis (без объявленных состояний) запрещено.

---

## Состояния

### `NoteSelection` — выбор нот и рекомендации

```
[без состояния]
      │  /start или команда «Подобрать духи»
      ▼
choosing_categories   ← пользователь выбирает категории (мультивыбор)
      │  нажать «Далее →»
      ▼
choosing_notes        ← пользователь выбирает конкретные ноты (мультивыбор)
      │  нажать «Показать подборку»
      ▼
viewing_results       ← бот показывает карточки парфюмов (пагинация)
      │  нажать «В избранное» / «Начать заново» / команда /menu
      ▼
[без состояния]
```

### `FavoritesMenu` — управление избранным

```
[без состояния]
      │  команда «Моё избранное»
      ▼
viewing_list          ← список избранного (пагинация)
      │  нажать «Убрать» / «Вкл./Откл. уведомления» / «Назад»
      ▼
[без состояния]
```

## Данные контекста FSM

Данные, хранимые в FSM-контексте (`state.update_data / state.get_data`):

| Ключ | Тип | Где устанавливается | Описание |
|------|-----|---------------------|----------|
| `selected_categories` | `list[int]` | `choosing_categories` | id выбранных категорий |
| `selected_notes` | `list[int]` | `choosing_notes` | id выбранных нот |
| `results_page` | `int` | `viewing_results` | текущая страница результатов (0-based) |
| `results` | `list[int]` | `viewing_results` | id парфюмов из текущей подборки |
| `favorites_page` | `int` | `viewing_list` | текущая страница избранного |

**Правило**: данные контекста не должны содержать полных объектов — только идентификаторы.
Загрузка данных из БД происходит при рендеринге каждого экрана.

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | приветствие; сброс текущего состояния; начало нового сеанса |
| `/menu` | главное меню (кнопки: «Подобрать духи», «Моё избранное») |
| `/favorites` | открыть избранное напрямую |
| `/help` | краткая справка |

## Клавиатуры

Все клавиатуры объявляются функциями-строителями в `perfume_bot/bot/keyboards.py`.
Хардкод текста кнопок в обработчиках запрещён.

| Функция | Возвращает | Описание |
|---------|-----------|----------|
| `categories_kb(categories, selected)` | `InlineKeyboardMarkup` | чекбоксы категорий |
| `notes_kb(notes, selected)` | `InlineKeyboardMarkup` | чекбоксы нот |
| `perfume_card_kb(perfume_id, in_favorites)` | `InlineKeyboardMarkup` | «В избранное» / «Купить» |
| `results_nav_kb(page, total)` | `InlineKeyboardMarkup` | пагинация результатов |
| `favorites_item_kb(favorite_id, notify)` | `InlineKeyboardMarkup` | управление позицией |
| `favorites_nav_kb(page, total)` | `InlineKeyboardMarkup` | пагинация избранного |
| `main_menu_kb()` | `ReplyKeyboardMarkup` | главное меню |
